递归的实现、特性以及思维要点
    递归-循环
    通过函数体来进行循环
    盗梦空间 特点：
        向下进入到不同的梦境中；向上又回到原来一层
        通过声音同步回到上一层
        每一层的环境和周围的人都是一份拷贝、主角等几个人穿越不同层级的梦境

    代码模板
        函数 (level,param){
            1、递归终结条件
            2、处理当前层逻辑 (level,param)
            3、下探到下一层   (level+1,param)
            4、清理当前层
        }
    思维要点：
        1、不要人肉进行递归
        2、找到最近最简的子问题
        3、数学归纳法思维

    分治和回溯
        分治
            1、递归终止条件
            2、逻辑处理
            3、下探到下层，继续解决子问题
            4、合并子问题的结果
        回溯
            试错的步骤，不满足就会退到上一层
